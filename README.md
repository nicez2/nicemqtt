# nicemqtt

####  1.在 nicemqtt/src/main/resources/application.yml 中指定productId 和 accessKey，指定后端项目运行在localhost的8080端口。
####  2.解压目录中的 nice-mqtt.zip，运行其中的 nice-mqtt.exe。
####  3.在程序中填写相关数据，并上传device列表文件，格式为xls。内容与onenet模板文件保持一致即可。
####  4.上传成功后，即可跳转到结果看板。
